# Workflows

This is just an example workflow specifications.
---

# Post Lifecycle

Draft → Pending Review → Approved
Draft → Pending Review → Rejected → Draft

---

# Approval Flow

1. Author or Admin creates draft.
2. Author submits for review.
3. Admin is auto-approved.
3. Admin reviews Author submitted posts:
   - Approve → visible publicly
   - Reject → editable by author

---

# Editing Rules

If approved post is edited:
→ Status resets to draft
→ Must be re-approved
