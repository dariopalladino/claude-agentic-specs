# Overall rules

This file gives you the basic foundational info to properly use this repository.

- All Agents are defined in AGENTS.md and detailed in .claude/agents folder, use agents.
- All specifications are in the .spec/ folder and detailed in the .spec/README.md file.
- All tasks with priorities can be found in the TODO.md file in the root.

The application is located in app/ folder where you can also find the git repository app/.git

### Create a new branch using the following command if needed:
```
git checkout -b f-<branch-name>
```

### To enable the virtual environment, you can run:
```
source .venv312/bin/activate
```

## Use SKILLS
Always use caveman skill to reduce tokens consumption

## README.md is the Core documentation
The README.md is the main document to understand the picture and make the right decision. Always keep the README.md file up-to-date with the latest changes.
